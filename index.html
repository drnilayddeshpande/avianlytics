How to Use Avianlytics

Welcome to Avianlytics. This tool helps you record daily/weekly data, calculates performance metrics, and runs immediate statistical analyses.

1
Create & Set Up Your Experiment
Go to the Trial Setup tab. Select if the trial is for Broilers or Layers, input the birds per group, set up the specific groups, treatments, and replicates, and click Save Configuration.
2
Daily Data Entry (Feed Offered, Layer Extras)
In the Data Entry tab, input daily feed offered. If you run a Layer experiment, you will additionally see options to enter daily egg counts and mortalities to dynamically adjust HDEP.
3
Weekly Records
At the end of each week, enter the average Body Weight (BW) and any Residual Feed (leftovers). The app calculates Feed Intake, FCR, and Layer production outputs automatically.
4
Analyze Results
Visit the ANOVA Analysis tab to run One-Way ANOVA tests comparing your treatments across parameters including Egg Production indices (HDEP/HHEP).

Data Storage & Privacy

This app handles data locally to ensure your privacy. Here is how your data is saved:

1. Automatic Local Storage (Workspaces)

Every experiment you create and every record you add is instantly saved in your web browser. You can close the tab and your separate experiments will still be here when you return.

2. Permanent Backup (Excel Export)

To create a permanent, sharable backup of a trial, make sure it is your active experiment, then click "Export Active Experiment to Excel" in the Dashboard tab.
